## 🚀 Getting Started  

### Prerequisites  
- Python 3.8+  
- Git (optional)  

### Setup  
1. **Clone the repository**:  
   ```bash
   git clone https://github.com/AfrinJahanEra/Python_Mini_Project.git
   cd Python_Mini_Project
   ```  
2. **Install dependencies** (if needed):  
   ```bash
   pip install -r requirements.txt 
   ```  
3. **Run any project**:  
   ```bash
   python projects/quickmath.py  
   ```
